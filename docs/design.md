# 1. Design Philosophy

Afghan Tappeti is a luxury brand specializing in authentic hand-knotted Oriental and Persian rugs.

The website should communicate:

- Luxury
- Heritage
- Trust
- Craftsmanship
- Elegance
- Timeless Beauty

The website must never feel like a generic ecommerce template.

Visitors should immediately feel they are browsing a premium collection.
